# Skill: QA Checklist (35 Points)

The definitive quality standard for SmolGame.

### Technical (1-10)
1. `touch-action: none` on canvas.
2. `localStorage` try/catch wrapper.
3. `requestAnimationFrame` loop.
4. Frame-independent `dt` logic.
5. Responsive `resize()` handler.
6. NO `location.reload()`.
7. NO absolute paths.
8. NO `window.parent` access.
9. Zero external dependencies.
10. Valid HTML5/ES6 syntax.

### UX & Mobile (11-20)
11. Default Portrait (9:16).
12. Bottom-aligned controls.
13. Large touch targets (>40px).
14. Visual Joystick/Swipe feedback.
15. NO text selection (`user-select: none`).
16. Screen shake on damage/impact.
17. Clear "START" and "RETRY" flow.
18. High Score persistence.
19. Fast loading (<1s).
20. No hover-state dependencies.

### Juice & Art (21-30)
21. Multi-layer parallax.
22. Particle explosions.
23. Dynamic Neon Glow.
24. Procedural Audio SFX.
25. Squash & Stretch animations.
26. Vignette/Scanlines filters.
27. Curated color palette.
28. Floating combat text.
29. Object trails (ghosting).
30. Smooth UI transitions.

### Meta (31-35)
31. Clean code organization.
32. Self-playing Demo Mode.
33. Dynamic Difficulty increase.
34. HUD rendered on Canvas.
35. Explicit Start tap to unlock audio.
