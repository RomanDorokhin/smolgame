# Genre: Snake

Traditional grid-based movement with modern "juice".

## 1. Mechanics
- Grid system: Map the canvas to a virtual grid (e.g., 20x20).
- Input: Queue the next direction to prevent 180-degree turns.
- Growth: Increase tail length when food is eaten.

## 2. Advanced Juice
- **Trail:** Add a faint glowing trail behind the snake.
- **Wobble:** Make the head wobble slightly based on speed.
- **Particle Pop:** Spawn particles when food is eaten.
- **Dynamic Speed:** Increase movement frequency as the snake grows.
