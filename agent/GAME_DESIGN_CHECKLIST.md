# SmolGame: Ultimate Game Quality Checklist

Этот документ определяет стандарты качества для игр на платформе SmolGame. Игры, не соответствующие "Juice" стандартам, будут отправлены агентом на доработку.

## 1. Технический фундамент (Обязательно)
- [ ] **Touch First:** Полная поддержка тач-событий (`pointerdown`).
- [ ] **No Scroll:** `touch-action: none` для предотвращения скролла страницы.
- [ ] **Safe Storage:** Весь `localStorage` обернут в `try/catch`.
- [ ] **Restart Loop:** Рестарт без перезагрузки страницы (`location.reload` запрещен).

## 2. "Juice" и Визуальные эффекты (Уровень качества)
- [ ] **Параллакс (Parallax):** Минимум 3 слоя фона (дальний, средний, ближний), движущихся с разной скоростью.
- [ ] **Screen Shake:** Тряска экрана при столкновениях, прыжках или взрывах.
- [ ] **Частицы (Particles):** Эффектные взрывы или брызги при смерти игрока или сборе предметов.
- [ ] **Трейлы (Trails):** Визуальный след за игроком или быстрыми объектами.
- [ ] **Свечение (Neon Glow):** Использование `shadowBlur` для создания неонового стиля.
- [ ] **Атмосфера:** Дополнительные слои (дождь, туман, помехи/scanlines, виньетка).

## 3. Геймплей и Прогрессия
- [ ] **Стейт-машина:** Четкие экраны Intro -> Game -> Game Over.
- [ ] **Динамическая сложность:** Скорость игры или частота препятствий растет со временем.
- [ ] **HUD:** Стильное отображение счета и рекорда прямо на Canvas (не через DOM).
- [ ] **Вторичные механики:** Например, двойной прыжок, рывок или комбо-система.

## 4. Звуковое сопровождение
- [ ] **Процедурный звук:** Использование Web Audio API для генерации SFX (прыжок, смерть, очки).
- [ ] **Инициализация:** Звук разблокируется только после первого тапа игрока.
