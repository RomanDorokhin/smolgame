Что это?

SmolGame — лента мини-игр в Telegram в духе TikTok: листаешь карточки, заходишь в игру по кнопке, на автора можно подписаться. Сейчас без обязательной рекламы перед игрой (никаких «смотри 10 сек, потом играй»). Свайп вверх / вниз — другая игра; «Играть» — полный экран, выход — «← Лента».

SmolGame is a mini-game feed in Telegram, TikTok-style: swipe cards, tap Play, follow creators. No mandatory ad before play right now. Swipe up / down for another game; Play is fullscreen, exit with ← Feed.

Как листать ленту

Свайп по экрану (не по кнопкам сбоку и снизу) — следующая или предыдущая игра.

  • Вверх — следующая игра
  • Up — next game
  • Вниз — предыдущая
  • Down — previous
  • «Играть» — только игра, без лайков и табов; «← Лента» — выход
  • Play — game only, no likes/tabs; ← Feed — exit
  • В режиме игры свайп ленты отключён
  • Feed swipe is off while playing

Step-by-step guide — form above.

1 Создай игру с ИИ

Зайди на claude.ai или chatgpt.com — оба бесплатны. Напиши (можно скопировать):

Напиши мне простую браузерную игру на HTML + JavaScript.
Один файл, без внешних библиотек.
Жанр: [змейка / тетрис / кликер / своё]
Игра должна работать на мобильном телефоне.

ИИ выдаст код — скопируй целиком. Не понравилось — попроси переделать.

2 Выложи на GitHub Pages

Бесплатный хостинг от GitHub, без карты и серверов.

  1. github.com → регистрация (если нет)
  2. + → New repository, имя например my-game, Public → Create
  3. Add file → Create new file → имя index.html → вставь код → Commit changes
  4. Settings → Pages → Branch: main → Save

Через 1–2 мин ссылка вида https://твой-ник.github.io/my-game — это URL игры.

3 Скопируй ссылку

В Settings → Pages зелёная плашка: Your site is live at https://… — скопируй её. Подожди 1–3 мин после публикации, если не открывается сразу.

1 Build a game with AI

Open claude.ai or chatgpt.com. Prompt (you can copy):

Make a simple browser game in HTML + JavaScript.
Single file, no external libraries.
Genre: [snake / tetris / clicker / your idea]
Must work on a mobile phone.

Copy the full code. Ask for changes if needed.

2 Publish on GitHub Pages

Free hosting from GitHub, no card or servers.

  1. github.com → sign up
  2. + → New repository, name it e.g. my-game, Public
  3. Add file → Create new file → index.html → paste code
  4. Settings → Pages → Branch: main → Save

In 1–2 mins your link is https://username.github.io/my-game.

3 Copy the link

In Settings → Pages find the green banner — copy it. Wait 1–3 mins if it doesn't open immediately.
