# SmolGame 2.0 — AI-Driven Game Universe

**SmolGame** — это не просто лента игр. Это экосистема, где грань между игроком и разработчиком стирается. С помощью **God Mode Agent**, любой пользователь может создать высококачественную, «сочную» игру за считанные минуты прямо в Telegram.

## ⚡️ Философия "God Mode"
Мы отказались от простых HTML-заглушек. Наш агент работает в три фазы, чтобы гарантировать коммерческое качество:
1.  **Interviewer**: Вытягивает из тебя идею и превращает её в дизайн-документ с упором на атмосферу и "Juice".
2.  **Engineer**: Пишет чистый, производительный код с использованием современных паттернов (State Machines, Particle Systems, Parallax).
3.  **Brutal QA**: Проверяет код на соответствие «стандартам сочности» (Juice Score > 85%). Если в игре нет тряски экрана, неона или частиц — QA заставляет переделывать всё с нуля.

## 💎 Juice Standards (Наши требования к играм)
Каждая игра, созданная в SmolGame, обязана иметь:
- **Atmosphere**: Глубокие фоны, неоновое свечение (Bloom/Glow), динамическое освещение.
- **Feedback**: Screen Shake (тряска экрана), частицы (Particles) при каждом действии, плавные анимации.
- **Mobile First**: Идеальная работа с Touch-событиями (`pointer events`), адаптивный вьюпорт и `touch-action: none`.
- **Safe State**: Использование `safeStorage` для защиты от вылетов в WebView Telegram.

## 🏗️ Архитектура платформы
Проект разделен на три ключевых модуля:

### 1. 🤖 Agent Smol (Core)
Сердце системы на React + Vite. Оркестратор LLM-моделей, который превращает промпты в рабочие игровые движки.
- **Location**: `/agent`
- **Engine**: OpenGame Core v4.4
- **Features**: 3-phase generation, Brutal QA, Juice Analyzer.

### 2. 🎬 Studio & Editor
Интегрированная среда разработки (IDE) прямо в приложении.
- **Live Preview**: Мгновенное тестирование игры в мобильном вьюпорте.
- **Iteration Loop**: Возможность попросить агента доработать любой аспект («сделай босса сложнее», «добавь больше неона»).
- **One-Click Deploy**: Публикация в глобальную ленту через GitHub Pages одной кнопкой.

### 3. 📱 Discovery Feed
Вертикальная лента для вирального распространения игр.
- **Swipe-to-Play**: Бесшовное переключение между играми.
- **Viral Growth**: Каждая игра — это отдельный репозиторий на GitHub, готовый к шерингу.

## 🛠 Технологический стек
- **Frontend**: React, TypeScript, Tailwind CSS, Lucide Icons.
- **Game Logic**: Pure HTML5 Canvas / WebGL (высочайшая производительность).
- **Backend**: Cloudflare Workers + D1 (Edge Computing).
- **AI**: Multi-provider failover (Groq, Gemini, OpenRouter, DeepSeek).

## 🚀 Как запустить свою версию
1.  Склонируй репозиторий.
2.  В папке `agent` создай `.env` (или используй интерфейс настроек в приложении для ввода API ключей).
3.  `npm install && npm run dev` — и ты в режиме создателя.
